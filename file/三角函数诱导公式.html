三角函数诱导公式

诱导公式一

对于任意整数 \(k\) ，有：

\(\sin(\alpha + 2k\pi) = \sin\alpha\)，\(k\in Z\)；

\(\cos(\alpha + 2k\pi) = \cos\alpha\)，\(k\in Z\)；

\(\tan(\alpha + 2k\pi) = \tan\alpha\)，\(k\in Z\)。

诱导公式二

\(\sin(\pi + \alpha) = -\sin\alpha\)；

\(\cos(\pi + \alpha) = -\cos\alpha\)；

\(\tan(\pi + \alpha) = \tan\alpha\)。

诱导公式三

\(\sin(-\alpha) = -\sin\alpha\)；

\(\cos(-\alpha) = \cos\alpha\)；

\(\tan(-\alpha) = -\tan\alpha\)。

诱导公式四

\(\sin(\pi - \alpha) = \sin\alpha\)；

\(\cos(\pi - \alpha) = -\cos\alpha\)；

\(\tan(\pi - \alpha) = -\tan\alpha\)。

诱导公式五

\(\sin(\frac{\pi}{2} - \alpha) = \cos\alpha\)；

\(\cos(\frac{\pi}{2} - \alpha) = \sin\alpha\)。

诱导公式六

\(\sin(\frac{\pi}{2} + \alpha) = \cos\alpha\)；

\(\cos(\frac{\pi}{2} + \alpha) = -\sin\alpha\)。

诱导公式七

\(\sin(\frac{3\pi}{2} - \alpha) = -\cos\alpha\)；

\(\cos(\frac{3\pi}{2} - \alpha) = -\sin\alpha\)。

诱导公式八

\(\sin(\frac{3\pi}{2} + \alpha) = -\cos\alpha\)；

\(\cos(\frac{3\pi}{2} + \alpha) = \sin\alpha\)。

诱导公式证明

诱导公式一证明：

在单位圆中，角\(\alpha + 2k\pi\)（\(k\in Z\)）与角\(\alpha\)的终边相同。根据三角函数定义，终边相同的角的同名三角函数值相等，所以\(\sin(\alpha + 2k\pi) = \sin\alpha\)，\(\cos(\alpha + 2k\pi) = \cos\alpha\)，进而\(\tan(\alpha + 2k\pi)=\frac{\sin(\alpha + 2k\pi)}{\cos(\alpha + 2k\pi)}=\frac{\sin\alpha}{\cos\alpha}=\tan\alpha\)。

记忆：转了一整圈（360°）不变，再转也一样。（2k是偶数）

诱导公式二证明：

设角\(\alpha\)终边上一点\(P(x,y)\)，\(r=\sqrt{x^{2}+y^{2}}\)，则\(\sin\alpha=\frac{y}{r}\)，\(\cos\alpha=\frac{x}{r}\)。角\(\pi+\alpha\)的终边与角\(\alpha\)终边关于原点对称，角\(\pi+\alpha\)终边上一点\(P'(-x,-y)\)，\(r'=\sqrt{(-x)^{2}+(-y)^{2}}=\sqrt{x^{2}+y^{2}}=r\)。所以\(\sin(\pi+\alpha)=\frac{-y}{r}=-\sin\alpha\)，\(\cos(\pi+\alpha)=\frac{-x}{r}=-\cos\alpha\)，\(\tan(\pi+\alpha)=\frac{-y}{-x}=\tan\alpha\)。

记忆：我们初中有句记忆口诀，“关于谁，谁不变；关于原点都改变”，在这里同样适用，你得到坐标后根据定义可得结果，所以你记忆时，仅需这句话“关于原点都改变”（转180°（\(\pi\)）），cos是x坐标，sin是y坐标，具体请见课本，这里不过多赘述。下同。当然你想“奇变偶不变，符号看象限”也行，不记也行，看你自己。

诱导公式三证明：

设角\(\alpha\)终边上一点\(P(x,y)\)，\(r=\sqrt{x^{2}+y^{2}}\)，则\(\sin\alpha=\frac{y}{r}\)，\(\cos\alpha=\frac{x}{r}\)。角\(-\alpha\)的终边与角\(\alpha\)终边关于\(x\)轴对称，角\(-\alpha\)终边上一点\(P''(x,-y)\)，\(r''=\sqrt{x^{2}+(-y)^{2}}=\sqrt{x^{2}+y^{2}}=r\)。所以\(\sin(-\alpha)=\frac{-y}{r}=-\sin\alpha\)，\(\cos(-\alpha)=\frac{x}{r}=\cos\alpha\)，\(\tan(-\alpha)=\frac{-y}{x}=-\tan\alpha\)。

记忆：“关于谁谁不变”这里是关于x轴对称，x坐标不变，y坐标变，以此记忆即可。

诱导公式四证明：

设角\(\alpha\)终边上一点\(P(x,y)\)，\(r=\sqrt{x^{2}+y^{2}}\)，则\(\sin\alpha=\frac{y}{r}\)，\(\cos\alpha=\frac{x}{r}\)。角\(\pi-\alpha\)的终边与角\(\alpha\)终边关于\(y\)轴对称，角\(\pi-\alpha\)终边上一点\(P'''(-x,y)\)，\(r'''=\sqrt{(-x)^{2}+y^{2}}=\sqrt{x^{2}+y^{2}}=r\)。所以\(\sin(\pi-\alpha)=\frac{y}{r}=\sin\alpha\)，\(\cos(\pi-\alpha)=\frac{-x}{r}=-\cos\alpha\)，\(\tan(\pi-\alpha)=\frac{y}{-x}=-\tan\alpha\)。

记忆：“关于谁谁不变”这里是关于y轴对称，y坐标不变，x坐标变，以此记忆即可。

诱导公式五证明：

在单位圆中，设角\(\alpha\)终边与单位圆交点坐标为\( (x,y)\)，则\(\sin\alpha = y\)，\(\cos\alpha = x\)。角\(\frac{\pi}{2}-\alpha\)终边与角\(\alpha\)终边关于直线\(y = x\)对称，其终边与单位圆交点坐标为\((y,x)\)，所以\(\sin(\frac{\pi}{2}-\alpha)=x=\cos\alpha\)，\(\cos(\frac{\pi}{2}-\alpha)=y=\sin\alpha\)。

记忆：反函数嘛，x和y互换嘛，你证全等（一眼洞穿）嘛，记忆图。

诱导公式六证明：

由诱导公式五\(\sin(\frac{\pi}{2}-\alpha)=\cos\alpha\)，\(\cos(\frac{\pi}{2}-\alpha)=\sin\alpha\)，将\(\alpha\)换为\(-\alpha\)可得\(\sin(\frac{\pi}{2}+ \alpha)=\cos(-\alpha)=\cos\alpha\)，\(\cos(\frac{\pi}{2}+ \alpha)=\sin(-\alpha)=-\sin\alpha\)。

记忆：根据公式五“反函数嘛，x和y互换嘛，你证全等（一眼洞穿）嘛，记忆图。”看图，关于y轴对称。课本有。

诱导公式七证明：

由诱导公式五\(\sin(\frac{\pi}{2}-\alpha)=\cos\alpha\)，\(\cos(\frac{\pi}{2}-\alpha)=\sin\alpha\)，将\(\alpha\)换为\(\pi-\alpha\)可得\(\sin(\frac{3\pi}{2}-\alpha)=\sin(\pi + (\frac{\pi}{2}-\alpha))=-\sin(\frac{\pi}{2}-\alpha)=-\cos\alpha\)，\(\cos(\frac{3\pi}{2}-\alpha)=\cos(\pi + (\frac{\pi}{2}-\alpha))=-\cos(\frac{\pi}{2}-\alpha)=-\sin\alpha\)。

诱导公式八证明：

由诱导公式六\(\sin(\frac{\pi}{2}+ \alpha)=\cos\alpha\)，\(\cos(\frac{\pi}{2}+ \alpha)=-\sin\alpha\)，将\(\alpha\)换为\(\pi+\alpha\)可得\(\sin(\frac{3\pi}{2}+ \alpha)=\sin(\pi + (\frac{\pi}{2}+ \alpha))=-\sin(\frac{\pi}{2}+ \alpha)=-\cos\alpha\)，\(\cos(\frac{3\pi}{2}+ \alpha)=\cos(\pi + (\frac{\pi}{2}+ \alpha))=-\cos(\frac{\pi}{2}+ \alpha)=\sin\alpha\)。
